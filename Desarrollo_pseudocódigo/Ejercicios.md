Se usarán las variables 'Día', 'Mes', 'Año', 'Día_actual', 'Mes_actual', 'Año actual' y 'Edad'.

```
Inicio 
    Leer 'Día_actual' 'Mes_actual' 'Año_actual'

    Definir 'Edad' como ('Año_actual'-'Año')  
    Si 'Mes_actual'>'Mes' entonces  
        Si 'Día_actual'>'Día' entonces  
            Print Edad
        Si no 
            Print 'Edad'-1
        cerrar si
    cerrar si

        Si 'Día_actual'='Día' entonces

Fin
```
